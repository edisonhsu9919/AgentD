📄 PDF 翻译器

基于 Ollama 的智能 PDF 文档翻译系统

📋 项目概述

PDF 翻译器是一个 开源的 Python 应用程序，能够将 PDF 文件转换为 Markdown 格式，并通过本地 LLM 模型（via Ollama）将其翻译成中文。该项目提供了 CLI 和 Web UI 两种使用方式。

🔄 核心处理流程

1
📄 PDF 解析
使用 marker 库将 PDF 转换为结构化的 Markdown
→
2
✂️ 智能分段
按段落分割，符合字符数限制要求
→
3
🌐 AI 翻译
通过 Ollama API 调用本地模型翻译
→
4
📝 合并输出
重组翻译块，生成最终文档

🔄

按自然段落分割，支持 min/max 字符限制，优化翻译质量

🛠️ 技术栈

Python 3 Marker (PDF 解析) Streamlit (Web UI) Requests (HTTP 客户端) Ollama (LLM 后端) Qwen2.5 (默认模型)

🏗️ 项目架构

📦 converter/ - 转换模块

  • pdf_to_md.py: PDF → Markdown 转换器，使用 marker 的 PdfConverter
  • splitter.py: 智能文本分段器，基于双换行符分割段落，支持 min/max 限制合并

🌐 translator/ - 翻译模块

  • llm_api.py: Ollama API 集成，POST 到 /api/generate 端点
  • 默认模型: qwen2.5:1.5b @ http://localhost:11434

🚀 应用入口

  • main.py: CLI 模式入口，支持命令行参数
  • streamlit_ui.py: Web UI 模式，提供可视化界面
  • run.sh: 启动脚本，配置端口和地址

🚀 使用方法

方式一：命令行模式 (CLI)

python main.py \ --input_pdf testfiles/medical_case_report.pdf \ --output_dir outputs \ --min_chars 100 \ --max_chars 1000

方式二：Web UI 模式 (推荐)

# 方法 1: 使用启动脚本 ./run.sh # 方法 2: 直接运行 streamlit streamlit run streamlit_ui.py \ --server.port=8501 \ --server.address=0.0.0.0

启动后访问: http://localhost:8501

🌐 启动 Web UI 📖 查看文档

⚙️ 配置参数说明

参数名称 默认值 说明
min_chars 100 每个文本块的最小字符数
max_chars 1000 每个文本块的最大字符数
model qwen2.5:1.5b 使用的 Ollama 模型名称
url http://localhost:11434/api/generate Ollama API 地址
output_dir outputs 输出文件保存目录

📁 文件命名规则

系统使用 UUID 前缀避免文件名冲突：

输入文件: {uuid}_{original_filename}.pdf Markdown: {uuid}_{original_filename}.md 翻译结果: {uuid}_{original_filename}_translated.md # 示例: 41dca899_Corporate_Governance.pdf 41dca899_Corporate_Governance.md 41dca899_Corporate_Governance_translated.md
